# Anudip Foundation - Java Programming Exercises

This repository contains Java programming exercises and implementations from the Anudip Foundation training program.

## Contents

- **Pattern Programs**: Various pattern printing exercises including triangles, rectangles, and number patterns
- **OOP Concepts**: Object-oriented programming implementations with classes, objects, constructors, and methods
- **Basic Programs**: Fundamental Java programs covering:
  - Arithmetic operations
  - Control flow (if-else, loops)
  - Mathematical operations (factorial, leap year, palindrome)
  - Input/output handling
  - Type casting

## Programs Included

- Hello World
- Addition Methods
- Even/Odd Number Checker
- Factorial Calculator
- Leap Year Checker
- Maximum of Three Numbers
- Palindrome Checker
- Pattern Programs (Triangle, Rectangle, Inverted Triangle)
- Constructor & Methods Examples
- Multiple Class and Object Implementations
- Static Method Examples
- Type Casting Examples

## Technologies

- Java
- Object-Oriented Programming

## Learning Objectives

This collection demonstrates core Java programming concepts learned during the Anudip Foundation training, focusing on:
- Basic syntax and programming logic
- Pattern recognition and implementation
- Object-oriented design principles
- Problem-solving with Java
